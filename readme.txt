Payzah is a third-party payment gateway that supports KNET & Credit Cards Transactions (VISA & MasterCard).

== Description ==
This plugin is provide the payment gatway service with Payzah, and which is support the KNET  & Credit Cards Transactions like Visa & Mastercard.

== Installation ==
First download the plugin from the WordPress plugin directory.
After go to the plugins setting from the admin panel and activate the plugin \"Payzah Payment Gateway for WooCommerce\".
Once plugin successfully installed, then go to the WooCommerce setting and click on the Payment Tab.
There will be show the \"TAJER BY PAYZAH\" payment gatway and click on the \"Manage\" button.
Once you enter the payment setting page, then there is an option to enable the payment gatway for the checkout page,.
You can enable it and then there is another main require field is \"Private key\".
Which will be provided by the \"payzah\"
You can send email \"info@payzah.com\" to get the private key.
After you need to enable the fields, for the checkout page billing forms, which is required fields.
If you enable the payzah payment gatway then you must be enable the checkout form fields from the payment gatway setting page. 
  

== Frequently Asked Questions ==

= How to get Private key for the payment gateway? =

You can send an email on \"info@payzah.com\', to get a private key for the payment gateway.

= How to enable the billing fields on checkout page? =

From the payment gateway setting page, you can enable the necessary required fields, then it will be visible for the billing fields on the checkout page.
